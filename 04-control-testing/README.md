# 04 — Control Testing

Prompts for designing walkthrough questions and writing up 
control deficiencies.

This folder will contain refined prompts for tasks such as:
- Designing walkthrough interview questions for a given control
- Writing up identified control deficiencies in standard format
- Drafting recommendations to management on deficiencies

Each prompt file documents the task, the prompt (v1 rough and 
v2 refined), example output, and a verification log noting what 
the AI got wrong.

# 01 — Research

Prompts for understanding accounting and auditing standards, 
comparing frameworks, and researching industry-specific audit risks.

This folder will contain refined prompts for tasks such as:
- Summarising IFRS / MFRS / ISA standards with verifiable references
- Comparing IFRS vs MFRS where they diverge
- Researching audit risks specific to an industry (e.g. property 
  developers under MFRS 15, banks under IFRS 9)

Each prompt file documents the task, the prompt (v1 rough and 
v2 refined), example output, and a verification log noting what 
the AI got wrong.

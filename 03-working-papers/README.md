# 03 — Working Papers

Prompts for drafting audit documentation: memos, walkthrough 
narratives, sampling rationale.

This folder will contain refined prompts for tasks such as:
- Drafting audit memos in standard working-paper format
- Writing up control walkthrough narratives
- Documenting sampling rationale and conclusions

Each prompt file documents the task, the prompt (v1 rough and 
v2 refined), example output, and a verification log noting what 
the AI got wrong.
